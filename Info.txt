Что нужно для проекта:
1) Инсталяция pipenv (виртуальной среды),
чтобы необходимые модули устанавливать локально
2) Создание вирт среды и ее запуск (pipenv shell)
3) Инсталяция джэнго
4) Создание нового проекта (django-admin startproject base .)
5) запуск сервера (python manage.py runserver), все действия в проекте через файл manage.py
6) создание приложения (app) через команду python manage.py startapp 'name'
7) Создание суперюзера (админ) (python manage.py createsuperuser)
8) Создание новых миграций для всех зареганных приложений (python manage.py makemigrations)
9) Применение миграций (python manage.py migrate)
10) Model - это интерфейс взаимодействия с БДб.
    View - это функция вида, которая обрабатывает запрос от клиента и возвращает ответ клиенту.
    Template - это шаблоны, которые видит пользователь.
    Шаблон всегда взаимодействует с моделью через вид.
    А функция вида с БД через модели
11) tastypie - библиотека для создания API приложения
    API - это программный интерфейс взаимодействия сервера и клиента
12) Чтобы создать ключ API, нужно зарегать модуль tastypie в приложениях base.settings
    и применить миграции для приложения tastypie
